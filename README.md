# Arakathir - College Training & Placement Website

## 📌 Project Overview

**Arakathir** is a professional website for a college training company that provides:
- Technical training (programming, software, domain skills)
- Verbal training (communication, soft skills, interview preparation)
- **Unique selling point:** A separate personal counselor available during college working hours
- 24/7 doubt support for students
- Student-wise analysis reports for colleges
- Placement support and project guarantee
- Minimum budget with high quality training
